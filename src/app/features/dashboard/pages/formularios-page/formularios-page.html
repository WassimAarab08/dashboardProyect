Cuándo usar qué

  • ReactiveFormsModule: validaciones complejas, formularios grandes, tipado estricto, statusChanges.
  • FormsModule: formularios simples y rápidos, plantillas cortas.
  • updateOn: usa 'blur' para performance, 'submit' para validar al enviar.

Buenas prácticas

  • Tipa siempre los controles: FormControl<string>.
  • Usa validadores reutilizables y asíncronos para checks a servidor.
  • Muestra mensajes solo si touched && invalid.
  • Deshabilita el submit si form.invalid.
  • Aplica clases condicionales con ngClass o estados CSS ng-invalid.
